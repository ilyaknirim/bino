# Beaver Run - Telegram Mini App

Игра "Beaver Run" адаптированная для Telegram Mini App. Управляйте бобром, избегая препятствий!

## Особенности Telegram Mini App версии

- Интеграция с Telegram WebApp API
- Адаптированный интерфейс для Telegram
- Сохранение рекордов
- Возможность поделиться результатом
- Тактильная обратная связь (вибрация)
- Улучшенная графика и анимации

## Управление

- **Прыжок**: Пробел на клавиатуре, свайп вверх по экрану или кнопка "⬆"
- **Приседание**: Стрелка вниз на клавиатуре, свайп вниз по экрану или кнопка "⬇"

## Развертывание в Telegram

1. Загрузите файлы игры на ваш веб-сервер (должен быть доступен по HTTPS)
2. Зарегистрируйте ваше приложение в Telegram через [@BotFather](https://t.me/botfather):
   - Создайте бота командой `/newbot`
   - Получите токен бота
   - Настройте Mini App командой `/newapp`
3. В настройках Mini App укажите URL вашего приложения
4. Готово! Теперь можно запустить игру через Telegram

## Структура проекта

```
dino/
├── index.html           # Основной HTML-файл
├── js/
│   ├── main.js          # Основная логика игры и интеграция с Telegram
│   ├── game.js          # Игровой движок
│   ├── beaver.js        # Логика бобра
│   ├── obstacles.js     # Логика препятствий
│   └── telegram.js      # Интеграция с Telegram WebApp API
└── style.css            # Стили игры
```

## Требования к серверу

- HTTPS (обязательно для Telegram Mini App)
- Поддержка ES6 модулей
- Правильные MIME-типы для JS файлов (`application/javascript`)

## Советы по оптимизации

- Используйте CDN для статических ресурсов
- Оптимизируйте изображения и спрайты
- Включите сжатие gzip на сервере
